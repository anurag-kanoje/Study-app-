# StudyBuddy

StudyBuddy is an AI-powered learning companion application built with Next.js and integrated with mini-omni2 for advanced text and video summarization capabilities.

## Features

- Smart Notes
- Time Management
- Flashcards
- Collaborative Tools
- Study Assistance
- Exam Preparation
- Topic Search
- Multilingual Support
- Stress Management
- Text Summarization
- Video Summarization

## Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS
- Firebase
- mini-omni2 (for AI-powered summarization)

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Set up your environment variables in `.env.local`
4. Run the development server with `npm run dev`
5. Start the mini-omni2 server with `npm run start-omni`
