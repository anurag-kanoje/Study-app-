# StudyBuddy Deployment Guide

This guide will help you deploy the StudyBuddy application with all its features fully functional.

## Prerequisites

1. **Node.js and npm**: Version 18.x or higher
2. **Firebase Account**: For authentication, database, and storage
3. **Vercel Account**: For deployment
4. **Mini-Omni2 Server**: For AI capabilities
5. **Google Cloud Account**: For translation services (optional)

## Step 1: Set Up Firebase

1. Create a new Firebase project at [firebase.google.com](https://firebase.google.com)
2. Enable Authentication (Email/Password and Google Sign-in)
3. Create a Firestore Database
4. Set up Storage for file uploads
5. Get your Firebase configuration (API keys, etc.)
